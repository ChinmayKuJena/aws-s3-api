# AWS S3 Handlers API

This API is built with NestJS and provides a comprehensive set of endpoints for interacting with Amazon S3 (Simple Storage Service). It enables users to manage S3 buckets and objects through RESTful API calls, including operations like listing buckets, uploading files, downloading objects, deleting resources, and generating pre-signed URLs for secure access.

## Key Features

- **Bucket Management**: List and manage S3 buckets.
- **Object Operations**: Upload, download, and delete objects in buckets.
- **Signed URLs**: Generate temporary URLs for secure file access.
- **Authentication**: Supports AWS credentials for secure S3 interactions.
- **Error Handling**: Global exception filters for robust error management.

## Getting Started

1. Install dependencies: `npm install`
2. Configure AWS credentials in your environment.
3. Run the application: `npm run start:dev`

For more details, refer to the API documentation or the source code.
